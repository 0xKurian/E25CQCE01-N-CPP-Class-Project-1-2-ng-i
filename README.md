# E25CQCE01-N-CPP-Class-Project-1-2-ng-i
## Project Statement
[P1].Design and implement a C++ scientific calculator. 
It allows the user enter a numeric expression at the program prompt >>, calculate and show the the result, repeat this step until <esc> key is pressed. For example:
  '>>a=10<enter>'
  
  'a is defined as 10'
  
  '>>b=a*2+1<enter>'
  
  'b is evaluated as 21'
  
  '>>c=d+1<enter>'
  
  'ERROR:d has not defined yet.'
