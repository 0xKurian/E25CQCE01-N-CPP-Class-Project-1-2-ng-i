1. Quick Sort là gì?

Tổng quan về Quick Sort

Quick Sort là một thuật toán sắp xếp hiệu suất cao, hoạt động dựa trên chiến lược cực kỳ nổi tiếng: "Chia để trị" (Divide and Conquer).

Bước 1: Chọn Chốt (Pivot)

Thuật toán không sắp xếp ngay lập tức mà luôn bắt đầu bằng việc chọn ra một phần tử làm mốc.

Phần tử mốc này gọi là Chốt (Pivot).

Bước 3: Đệ quy (Recursion)

Cuối cùng, thuật toán liên tục gọi đệ quy chính nó để lặp lại quá trình trên các mảng con.

Quá trình lặp lại cho đến khi toàn bộ mảng được sắp xếp hoàn chỉnh (mảng con chỉ còn 1 phần tử).

2. Cách chọn Chốt (Pivot)

Thay vì chỉ dùng một cách cố định, dự án tích hợp hai chiến lược phân hoạch kinh điển để linh hoạt so sánh.

Chiến lược Lomuto

  • Chọn phần tử cuối cùng của mảng/mảng con làm Pivot.
  • Ưu điểm: Mã nguồn cực kỳ đơn giản và dễ cài đặt, dễ hiểu cho vòng lặp duyệt.
  • Đối tượng: Rất phù hợp cho người mới bắt đầu học thuật toán.

Chiến lược Hoare

  • Chọn phần tử đầu tiên của mảng/mảng con làm Pivot.
  • Đặc điểm: Phức tạp hơn về mặt tư duy con trỏ.
  • Ưu điểm: Tối ưu hơn trong thực tế nhờ giảm thiểu được số lần hoán đổi (swap) so với Lomuto.

3. Hàm Phân hoạch (Partition)

Cơ chế "trái tim" của thuật toán thay đổi ra sao?

Cơ chế Phân hoạch Lomuto

  • Dùng một biến i đánh dấu ranh giới các phần tử nhỏ hơn Pivot.
  • Duyệt qua đoạn [l, r-1] bằng một vòng lặp.
  • Mỗi khi gặp phần tử <= Pivot, dồn phần tử đó về nửa bên trái và tăng i.
  • Cuối cùng, đặt Pivot vào đúng vị trí ranh giới i.

Cơ chế Phân hoạch Hoare

  • Sử dụng hai con trỏ i (chạy từ trái sang) và j (chạy từ phải sang).
  • Dừng lại khi phát hiện cặp phần tử nằm sai vị trí (phần tử lớn bên trái, phần tử nhỏ bên phải Pivot).
  • Tiến hành hoán đổi (swap) chúng cho nhau.
  • Lặp lại liên tục cho đến khi hai con trỏ vượt qua nhau.

4. Độ phức tạp (Complexity)

Thời gian (Time Complexity)

Trường hợp tốt nhất & Trung bình: O(n log n)

Đạt được khi Pivot chia mảng thành hai nửa tương đối bằng nhau ở mỗi tầng của cây đệ quy.

Thời gian (Time Complexity)

Trường hợp xấu nhất: O(n²)

Xảy ra khi mảng đầu vào đã được sắp xếp sẵn (hoặc ngược lại) và thuật toán rơi vào tình huống phân hoạch bị lệch hoàn toàn (một bên 0 phần tử, bên kia n-1 phần tử).

Không gian (Space Complexity)

Trường hợp trung bình: O(log n)

Do thuật toán thao tác trực tiếp trên mảng gốc (in-place) và không cần mảng phụ. Bộ nhớ thiết yếu duy nhất là Ngăn xếp gọi hàm (Call Stack) để lưu vết đệ quy.

  • Quản lý an toàn dữ liệu:

Công việc hoàn thành (2/2)

  • Đa dạng hoá đệ quy:
    Cài đặt cả 2 cặp hàm độc lập: Partition_Lomuto và partition_Hoare.
  • Hiệu ứng Step-by-step:
    Lồng hàm printArray() sau mỗi lệnh swap() để in trực tiếp bước đổi chỗ trên Console.
  • Menu Console:
    Xây dựng giao diện nhập xuất tương tác cho người dùng chọn thuật toán minh hoạ.

1. Lớp & In mảng


#include <bits/stdc++.h>
using namespace std;

class QuickSortDemo {
private:
    vector<int> arr;

    // Hàm in mảng minh họa: [ 10, 20, 30 ]
    void printArray() {
        cout << "[ ";
        for (int i = 0; i < arr.size(); i++) {
            cout << arr[i] << (i < arr.size() - 1 ? ", " : "");
        }
        cout << " ]\n";
    }

2. Phân hoạch Lomuto


int Partition_Lomuto(int l, int r) {
    int pivot = arr[r];
    int i = l - 1;
    cout << "  -> Pivot: " << pivot << endl;

    for (int j = l; j < r; j++) {
        if (arr[j] <= pivot) {
            ++i;
            swap(arr[i], arr[j]);
            if (i != j) {
                printArray(); // Step-by-step in ra
            }
        }
    }
    ++i;
    swap(arr[i], arr[r]); // Đưa pivot về giữa
    return i;
}

Đệ quy Lomuto


void QuickSort_Lomuto(int l, int r) {
    if (l >= r) return; // Điều kiện dừng đệ quy
    
    cout << "\n[Step] Sorting " << l << " to " << r << endl;
    
    // 1. Phân hoạch mảng
    int p = Partition_Lomuto(l, r);
    
    // 2. Đệ quy 2 nửa
    QuickSort_Lomuto(l, p - 1);
    QuickSort_Lomuto(p + 1, r);
}

3. Phân hoạch Hoare


int partition_Hoare(int l, int r) {
    int pivot = arr[l];
    int i = l - 1, j = r + 1;

    while (1) {
        do { ++i; } while (arr[i] < pivot);
        do { --j; } while (arr[j] > pivot);
        
        if (i < j) {
            swap(arr[i], arr[j]);
            printArray(); // In bước hoán đổi Hoare
        } else {
            return j; // Trả về ranh giới
        }
    }
}

public:
    void inputData() {
        int n; cin >> n;
        arr.resize(n); 
        for (int i = 0; i < n; i++) {
            cin >> arr[i];
        }
    }

    void startSorting() {
        int choice;
        cout << "1. Lomuto\n2. Hoare\nChoice: ";
        cin >> choice;

        if (choice == 1) QuickSort_Lomuto(0, arr.size() - 1);
        else QuickSort_Hoare(0, arr.size() - 1);
    }
